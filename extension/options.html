sol browser — settings

this prototype experiences only the web apps you add below — taking in their visible text, never screenshots — and relays it into your local journal as its own browser stream. nothing is read until you add a site. the toolbar icon is the always-on status light; the on-page marker is optional.

added sites

adding a site asks chrome for permission to read just that site. if chrome removes access, sol pauses it — allow again any time here; remove forgets it.

off by default. your always-on signal is the toolbar icon, which reads on / paused / needs attention at a glance. turn this on to add an on-page marker too.
